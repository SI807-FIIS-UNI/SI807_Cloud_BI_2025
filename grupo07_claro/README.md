Proyecto Cloud BI 2025-II

## Primera Parte: Limpieza y Preparación de Datos

* Se introdujo la información cruda del dataset (`sot_historica.csv`).
* Se creó un script (`LittleTransformacion.py`) para poder capitalizar la información y hacer un reemplazo de valores nulos por "Sin Información".
* Limpieza del dataset y obtención de la data limpia (`sot_historico_limpio.csv`).

## Segunda Parte: Análisis y Visualización

* Realización de análisis exploratorio (EDA) con scripts en un Jupyter Notebook (`pruebas analisi.ipynb`).
